Google Cloud Setup
One-time, 5-minute setup. Your files never leave Google Drive.
1
Create a Google Cloud Project

Go to console.cloud.google.com → New Project → Name: "GPM Media Library"

2
Enable the Google Drive API

APIs & Services → Enable APIs → Search "Google Drive API" → Enable

3
Configure OAuth Consent Screen

APIs & Services → OAuth consent screen → External → App name: "GPM Media Library" → Scopes: drive.file + drive.readonly

4
Create OAuth Credentials

Credentials → Create OAuth client ID → Web application → Add your domain to Authorized JavaScript Origins. For local use, add http://localhost

5
Create the GPM Media Library folder

In Google Drive, create a Shared Drive called "GPM Media Library". Share with Eric (Viewer) and Liam (Contributor). App auto-creates subfolders on first run.

6
Paste Client ID and sign in

Copy the Client ID from step 4, paste it on the sign-in screen, click Sign In. Done.
